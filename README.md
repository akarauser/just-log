# just-log

This script provides a **custom JSON-based logging system** with terminal visualization for structured, machine-readable logs. It supports **rotating log files**, **session tracking**, and **rich terminal output** with color-coded severity levels.

The code prioritizes **modularity** (separate logger and viewer components) and **error resilience** (graceful log file handling). It leverages Python’s `logging` module with a **custom JSON formatter** and `rich` for terminal styling.

Ideal for applications requiring **structured logs**, **real-time monitoring**, and **cross-platform compatibility**.
